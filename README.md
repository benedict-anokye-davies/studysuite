# StudySuite

Open-source student career opportunity platform. Find jobs, internships, spring weeks, insight weeks, hackathons, and scholarships — personalized to your degree and goals.

## Status
In development. Planning phase complete.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS + shadcn/ui
- tRPC
- PostgreSQL (Supabase)
- Drizzle ORM
- Vercel
